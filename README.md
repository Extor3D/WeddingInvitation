# Wedding Invitation
Wedding invitation made in Godot

Link to play: https://extor3d.itch.io/invitacion-casamiento-de-yohe-y-ale

## Controls

A, Left: Move to the left
D, Right: Move to the right
W, Up, Spacebar: Jump
L, Ctrl: Shoot (Only Yohe)
